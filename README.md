# Workflow Repository

This repository contains the Kanboard intake and `handoff/` workflow for WorksOnMine.

## Included

- `kanboard_setup/` - Kanboard ticket fetch / claim / handoff tooling
- `handoff/` - active task briefs and decision notes
- `docs/teams/` - agent roles and workflow instructions
- `bin/audit.sh` - local audit hook used before commits

## Notes

- Application code lives in the product repository.
- Keep generated audit output out of git.
