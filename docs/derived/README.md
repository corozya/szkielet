# Dokumenty Wygenerowane

Dokumenty tworzone na żądanie orchestratora na podstawie source docs z projektu.

**Zasada:** Każdy dokument ma źródło (`Źródło:` w nagłówku). Agent tworzy raz, żeby nie powtarzać pracy.

Przykłady:
- `DEPLOYMENT_GUIDE.md` — przewodnik wdrażania (źródło: `apps/reczniki-haftowane/docs/teams/DEVOPS.md`)
- `API_REFERENCE.md` — dokumentacja API (źródło: `apps/reczniki-haftowane/docs/teams/BACKEND.md`)
