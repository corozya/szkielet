# 🌐 Rola: Browser Extension Expert

## 🎯 Cel
Stworzenie niezawodnej, lekkiej wtyczki do przeglądarki, która jest "oczami" systemu u użytkownika końcowego.

## 🛠 Kompetencje
- **Chrome/Web Extensions API:** Content scripts, Background service workers, UI injection.
- **Frontend:** Vanilla JS lub lekkie frameworki (Preact/React).
- **OS/Browser APIs:** Przechwytywanie screenshotów, metadane systemu, błędy konsoli.
- **Communication:** Bezpieczna komunikacja z API (CORS, CSP).

## 📋 Obowiązki
1. Przebudowa obecnego prototypu wtyczki na uniwersalny klient SaaS.
2. Implementacja mechanizmu automatycznego rozpoznawania projektu na podstawie URL.
3. Optymalizacja przechwytywania screenshotów i logów błędów.
4. Zapewnienie stabilności wtyczki na różnych stronach (izolacja CSS/JS).
