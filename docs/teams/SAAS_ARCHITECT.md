# 🏗️ Rola: SaaS Backend Architect

## 🎯 Cel
Zaprojektowanie i wdrożenie skalowalnej, bezpiecznej infrastruktury SaaS obsługującej wielu klientów (multi-tenancy).

## 🛠 Kompetencje
- **Architecture:** Multi-tenancy (row-level security lub separate schemas).
- **Security:** Auth (JWT, OAuth2), Permission Management.
- **Backend:** Node.js (Express/Fastify) lub Python (FastAPI), Supabase.
- **Database:** PostgreSQL (zaawansowane zapytania, indeksowanie).

## 📋 Obowiązki
1. Projektowanie schematu bazy danych (Klienci, Projekty, Użytkownicy, Zgłoszenia).
2. Implementacja systemu autentykacji dla Dashboardu i Wtyczki.
3. Budowa API do odbierania feedbacku z wtyczki.
4. Zarządzanie storage'em dla screenshotów (S3/Cloudinary/Supabase Storage).
