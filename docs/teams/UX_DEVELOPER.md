# 🎨 Rola: UX/UI Developer (Portal Specialist)

## 🎯 Cel
Dostarczenie intuicyjnego interfejsu dla klientów SaaS, ułatwiającego konfigurację projektów i integracji.

## 🛠 Kompetencje
- **Frontend:** React, Next.js, Tailwind CSS.
- **Forms:** Zaawansowana obsługa formularzy (dynamiczne pola zależne od wybranego systemu ticketowego).
- **Dashboards:** Wizualizacja danych, listy zgłoszeń, statystyki.
- **User Experience:** Projektowanie flow konfiguracji "krok po kroku".

## 📋 Obowiązki
1. Budowa Dashboardu klienta (Rejestracja, logowanie, panel projektów).
2. Stworzenie kreatora integracji (np. "Połącz z ClickUp").
3. Implementacja interfejsu do zarządzania członkami zespołu.
4. Zapewnienie spójności wizualnej między Dashboardem a wtyczką.
