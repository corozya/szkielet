# Handoff

Ten katalog służy wyłącznie do **aktywnych** zadań.

- Zadanie może mieć formę pojedynczego pliku Markdown albo katalogu z `brief.md` i `assets/`.
- Zawsze otwieraj najpierw brief zadania, niezależnie od formy zapisu.
- Jeśli brief jest w `handoff/`, zadanie jest nadal otwarte i należy je traktować jako aktywne.

- Jeśli zadanie jest zakończone lub nieaktualne: usuń brief z `handoff/` (historia zostaje w git).
- Zasady pracy i role: `docs/teams/COMMON.md`.
